# 音频播放功能

1- 生成音频文件
2- 实现播放按钮的功能
    播放（循环播放）
    停止播放
3- 先实现pc端，再实现手机端
4- 使用手机访问时默认时pc端的页面，最好能够自动跳转到手机端的页面。

## 背景

服务器环境：PHP 8.2.29 ✅ + Apache mod_php ✅ + JSON 扩展 ✅

**PHP 方案的优势**：Apache 原生支持 PHP，不需要反向代理，不需要守护进程，部署最简单 —— 上传文件即可运行。

## 架构

```
Apache (80端口)
  └── /var/www/html/learn/
       ├── index.html          ← 前端入口
       ├── api/                ← PHP API
       │    └── index.php      ← 路由分发
       ├── .htaccess           ← URL 重写规则
       ├── data/procesed/*.json  ← 文章数据
       ├── data/audio/*.mp3      ← 音频文件
       ├── css/
       └── js/
```

### URL 重写

#### [NEW] [.htaccess](file:///Users/kooreywang/myproject/learn/.htaccess)

Apache URL 重写规则：
- `/api/articles` → `api/index.php?action=list`
- `/api/articles/{id}` → `api/index.php?action=detail&id={id}`
- 其他请求正常走静态文件
