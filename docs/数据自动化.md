# 数据处理自动化流程

## 输入数据说明

- 英文原文.如2014_passage_cloze.md 
- 一行为一段.

## step1: 提取句子.
- 按照标点符号分句.
- 输出为json格式.
- 每个句子包含:
    - paragraph: 段落编号
    - sentence_id: 句子的全文编号
    - sentence_md5: 句子原文的MD5值，也用作音频文件名
    - sentence: 句子原文
    - audio_file: 句子的音频文件名
- 存储文件为 ./data/sentences/2014_passage_cloze.step1.md. 一行为句压缩后的json.如:{"paragraph": 1,"sentence_id": 1,"sentence_md5": "8b1a9953c4611296a827abf8c47804d7","sentence": "In response, supporters of financial-literacy education are moving with renewed enthusiasm.","audio_file": "data/audio/8b1a9953c4611296a827abf8c47804d7.mp3"}
- 特殊说明
    - ()问题.如：
    "Their clever paper asks givers and receivers to classify gifts from two perspectives: desirability (e.g. the cost of a coffee maker) and feasibility (e.g. the ease of the coffee maker)."
    应拆分为: 
    "Their clever paper asks givers and receivers to classify gifts from two perspectives: desirability (e.g. the cost of a coffee maker) and feasibility (e.g. the ease of the coffee maker)."
## step2: AI生成分析数据.【手动完成】
- 英文句子.
- 每句英文输出一个json数据.
- json包含:
```markdown
    | 字段 | 含义 | 内部结构 | 说明 |
    | :--- | :--- | :--- | :--- |
    | i | sentence_id | 整数 | 句子的全文编号 |
    | p | phonetic | 字符串 | 句子的音标 |
    | t | translation | 字符串 | 句子的译文 |
    | l | long_sentence_analysis | 字符串 | 句子成分分析 |
    | v | core_vocabulary | 数组，每项 [word, phonetic, meaning] | 核心词汇及音标、释义 |
    | ph | phrases | 数组，每项 [phrase, meaning] | 短语及释义 |
```

- 存储文件为 ./data/sentences/2014_passage_cloze.step2.md. 一行为句压缩后的json.如:
```json
{"i": 1, "p": "", "t": "", "l": "", "v": [["response", "rɪˈspɒns", "n. 回应，反应"], ["supporter", "səˈpɔːtə(r)", "n. 支持者"], ["financial", "faɪˈnænʃl", "adj. 金融的，财务的"], ["literacy", "ˈlɪtərəsi", "n. 读写能力；素养"], ["enthusiasm", "ɪnˈθjuːziæzəm", "n. 热情"]], "ph": [["in response", "作为回应"], ["financial literacy education", "金融知识教育"], ["with renewed enthusiasm", "带着新的热情"]]}
```
## step3: 合并数据
- 将step1和step2的数据合并.
- 检查原文-音标对齐函数是否正常.（页面展示时，需要句子的英文-音标对齐。）
    - 检查函数是否正常.若不正常，可以处理数据或优化函数。
    - 常见问题：
        - “63”数字导致音标对齐失败.
        - “-”号导致音标对齐失败.
        - “（中文注释）”中文注释导致音标对齐失败.
- 合并后的数据格式如下:
```json 
{
    "article_metadata": {
        "title": "article_file_name",
        "author": "Unknown",
        "source": "CET-4/6 2014"
    },
    "sentences": [
        {
            "paragraph": 1,
            "sentence_id": 1, //全文编号
            "sentence_md5": "原文的MD5值",
            "sentence": "原文",
            "phonetic": "音标",
            "translation": "译文",
            "audio_file": "音频文件路径",
            "analysis": {
                "long_sentence_analysis": "只有较长的句子才需要分析。用一段文字描述句子成分分析",
                "core_vocabulary": [
                    {
                        "word": "example",
                        "phonetic": "ıg'zæmpl",
                        "meaning": "n. 例子，范例"
                    }
                ],
                "phrases": [
                    {
                        "phrase": "This is",
                        "meaning": "这是...（句型介绍）"
                    }
                ]
            }
        }
    ]
}
```
- 生成./data/procesed/article_file_name.json

## step4: 生成音频文件.
- 根据./data/procesed/article_file_name.json生成句子音频文件
- 生成./data/audio/sentence_md5.mp3
